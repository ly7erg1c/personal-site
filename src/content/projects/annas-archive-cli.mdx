[Anna's Archive CLI downloader](https://github.com/ly7erg1c/annas-archive-cli) is a simple python based script which:
- Takes an ISBN13 as an input.
- Finds the corresponding .torrent file from the archive where that content is hosted.
- Utilizes libtorrent to download the content.
